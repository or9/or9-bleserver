# or9-bleserver
BLE server

## Getting Started
This application requires NodeJS. Ensure it's installed.  
### Install NodeJS  
```
sudo apt-get install build-essentials
wget https://nodejs.org/dist/v8.1.0/node-v8.1.0.tar.gz
tar -xvf node-v8.1.0.tar.gz
cd node-v8.1.0
./configure
make
sudo make install
```
